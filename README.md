# 네이버 API를 이용하여 영화 검색하기

## Setup
### 파일수정
* .env.development
  ```
  VUE_APP_OPTIONS_CLIENT_ID=your-client-id // 네이버 클라이언트 id 입력
  VUE_APP_OPTIONS_CLIENT_SECRET=your-client-secret // 네이버 클라이언트 scret 입력
  ```
### 모듈 설치
```
yarn
```

### Developerment

```
yarn serve
```
